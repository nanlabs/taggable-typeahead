Taggable Typeahead

A small library to wrap the Twitter typeahead.js component and add the taggable functionality.

Default Behaviour

Typeahead with the taggable feature.

$('example1').typeahead( { 
  prefetch: 'data.json'
}).taggable();

With Max Limit (5)

Set the max number of tags.

$('example2').typeahead( { 
  prefetch: 'data.json'
}).taggable( { 
  max: 5 
});

With Typeahead Template

Use typeahead template to display options with more information.

$('#example3').typeahead( { 
  prefetch: 'data2.json',
  valueKey: 'label',
  template: '<span><%=label%></span><i class="pull-right">(<%=usage%>)</i>',
  engine: engine
}).taggable( { 
  valueKey: 'label' 
});
